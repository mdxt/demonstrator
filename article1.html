Scrollspy in Bootstrap 4

Overview Details Additional

Overview

While scrolling through a long page of text, or anything with multiple sections, it is often hard to keep track of where you are.

That is where Scrollspy comes in. As hinted at by its name, it 'spies' on the scroll position of the page, and highlights an element in an unordered list that links to the current section of the page.

This unordered list is usually a nav or a list group. If you scroll down and up, you will see that you are actually learning about Scrollspy through Scrollspy. Very meta.

Details


    <div id="list-example" class="list-group">
    <a class="list-group-item list-group-item-action" href="#overview">Overview</a>
    <a class="list-group-item list-group-item-action" href="#details">Details</a>
    <a class="list-group-item list-group-item-action" href="#additional">Additional</a>
    </div>

Above is the code for the list group you see on the left. You can see that we have specified links for each section of the main text.

And finally, given below is the skeleton code for the section you are reading.


    <div data-spy="scroll" data-target="#list-example" data-offset="0" class="scrollspy-example">
      <h2 id="overview">Overview</h2>
      <p class="lead">=  </p>
      <h2 id="details">Details</h2>
        <pre><code>
        </code></pre>
      <p class="lead"></p>
      <p class="lead"></p>
      <h2 id="additional">Additional</h2>
      <p class="lead"></p>
    </div>

Here, data-spy="scroll" is added to indicate that this div is to be spied upon, data-target specifies the list to be targeted, and data-offset gives the number of pixels to offset from top when calculating position of scroll.

Note that we added 'id's to the sections corresponding to the links in the list group.

Additional

Scrollspy must be used on a Bootstrap nav component or list group.
